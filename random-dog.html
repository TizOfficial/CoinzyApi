🐶 Coinzy API - Zufälliges Hundebild

Diese API gibt ein zufälliges Hundebild zurück.

🔗 API-URL

/API/random-dog

Jetzt testen

📚 Code-Beispiele

➡️ BDFD

$jsonRequest[https://coinzyapi.com/API/random-dog;;image]

➡️ Discord.js

const fetch = require("node-fetch");
fetch("https://coinzyapi.com/API/random-dog")
  .then(res => res.json())
  .then(data => console.log(data.image));

➡️ Python

import requests
data = requests.get("https://coinzyapi.com/API/random-dog").json()
print(data["image"])
